# Rule Specs (v1)

Each rule spec in this directory follows the same structure:

1. Intent
2. Compatibility tag (`compatible`, `inspired`, `native`)
3. Algorithm summary
4. Config options and defaults
5. Default severity
6. Fix support and safety constraints
7. Known divergences (if any)
8. Complexity and performance notes
